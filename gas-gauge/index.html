Install in 5 Minutes

Even if you've never installed a Chrome extension before, we've got you.

1

Download the Zip File

Click the big "Download the Extension" button above. Your browser will download a file called NumberOneSonFreebieClaudeGasGaugeChromeExt-main.zip. You'll find it in your Downloads folder.

2

Unzip the File

Windows: Right-click the zip file → click "Extract All..." → click "Extract". A new folder will appear.
Mac: Just double-click the zip file. A folder will appear right next to it.

💡 Remember where this folder is — you'll need it in Step 4!
3

Open Chrome's Extension Page

Open Google Chrome. In the address bar at the top, type chrome://extensions and press Enter. This opens a special page where Chrome manages all your extensions.

💡 You can also get here by clicking the three dots ⋮ in the top-right corner → Extensions → Manage Extensions
4

Turn On "Developer Mode"

Look at the top-right corner of the extensions page. You'll see a toggle switch that says "Developer mode". Click it so it turns blue (on). Three new buttons will appear at the top.

⚠️ Don't skip this step — the next button won't show up without it!
5

Load the Extension

Click the "Load unpacked" button (top-left). A file picker window will open. Navigate to the folder you unzipped in Step 2. Open that folder, then select the folder inside it (the one with the files like manifest.json) and click "Select Folder".

💡 If you see an error, you might have selected the outer folder. Make sure you pick the inner folder that contains the actual files.
6

Pin It & Go! 🎉

The extension is now installed! Look for the puzzle piece icon 🧩 in Chrome's toolbar (top-right, next to the address bar). Click it, then click the pin icon 📌 next to "Claude Fuel Gauge" so it's always visible.

Now go to claude.ai — your fuel gauge will appear at the top! Visit Settings → Usage once to load your numbers.

💡 Click the ⛽ extension icon anytime to inject the gauge into all your open Claude tabs at once.

Questions You Might Have

Is it really free?
Yes, 100% free. No accounts, no subscriptions, no data collection. If you find it useful and want to show appreciation, there's a "Buy Me a Coffee" link. That's it.
Is it safe? What permissions does it need?
The extension can only see claude.ai — it has zero access to any other website. It stores your usage percentages locally on your computer. No data is ever sent to us or anyone else. The code is fully open source so you can read every line.
Do I need a Claude Pro or Max subscription?
The gauge reads from Claude's settings page, which shows usage for any paid plan (Pro, Max, Team). Free-tier users may not see usage percentages on their settings page.
Why does it say "Developer mode"? Is this a developer tool?
Nope! "Developer mode" is just how Chrome lets you install extensions that aren't from the Chrome Web Store. Thousands of people use this method every day. It's completely normal and safe.
The gauge shows dashes (—) instead of numbers.
Go to claude.ai → click on your name (bottom-left) → Settings → Usage. Wait a few seconds for the page to load. The gauge will pick up your numbers. After that, it remembers them.
How do I update to a new version?
Download the zip again, unzip it to the same folder (replace the old files), then go to chrome://extensions and click the little refresh ⟳ icon on the Claude Fuel Gauge card.
